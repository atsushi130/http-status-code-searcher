## Installation via Cargo
```
❯ cargo install http_status_code_searcher
```

## Usage
```
❯ http_status_code_searcher 200
200 OK
リクエストは成功し、レスポンスとともに要求に応じた情報が返される。ブラウザでページが正しく表示された場合は、ほとんどがこのステータスコードを返している。
```
